# Chat2Word

一键将 ChatGPT、Gemini、Claude、DeepSeek 对话转为可编辑的 Word 文档，完整保留格式，支持表格、数学公式、代码块等。适用于工作报告、知识整理、内容编辑等办公场景。

## 核心优势

- 🎯 **办公场景优化**: 导出的 Word 文档可直接编辑、批注、协作，无缝融入工作流
- ✅ **真正的 DOCX 格式**: 标准 Office Open XML 文档，完美兼容 Microsoft Word、WPS
- ✅ **数学公式支持**: LaTeX 公式自动转换为 Word 原生格式（OMML），可在 Word 中继续编辑
- ✅ **多平台支持**: ChatGPT、Gemini、Claude、DeepSeek 等主流 AI 平台全覆盖
- ✅ **格式完美保留**: 标题、粗体、斜体、代码块、引用、列表等格式一键转换
- ✅ **一键转换**: 页面悬浮按钮 + 浏览器扩展，双重快捷方式
- ✅ **粘贴即转**: 复制 Markdown 内容，粘贴即可生成 Word 文档
